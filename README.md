# Twitter Clone

A responsive Twitter/X clone built with HTML, CSS, and JavaScript, featuring a modern UI and dark mode support.

## Features

-  Responsive design that works on desktop and mobile
-  Dark mode toggle
-  Sidebar navigation with hover effects
-  Interactive tweet composing interface
-  Profile section with dropdown menu
-  Search functionality
-  News and trending topics widgets
-  "Who to follow" suggestions
-  Chatbot integration

## Technologies Used

- HTML5
- CSS3 (with CSS Variables for theming)
- JavaScript
- Font Awesome Icons
- Material Icons
- Responsive CSS Grid/Flexbox Layout

## Project Structure

```
├── index.html          # Main application page
├── style.css          # Main stylesheet
├── responsive.css     # Responsive design styles
├── script.js         # JavaScript functionality
├── chatbot.html      # Chatbot interface
└── chatbot.css       # Chatbot styles
```

## Key Components

1. **Navigation Bar**
   - Twitter logo
   - Main navigation menu
   - Tweet button
   - Profile section

2. **Main Feed**
   - Tweet composition box
   - Tweet feed with images and interactions
   - Interactive action buttons (Like, Retweet, etc.)

3. **Right Sidebar**
   - Search box
   - Premium subscription widget
   - "What's happening" news section
   - "Who to follow" suggestions

4. **Theme Support**
   - Light/Dark mode toggle
   - CSS variables for easy theme customization
   - Smooth theme transitions
